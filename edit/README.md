# 病棟マップエディタ

`edit/index.html`をブラウザで開くだけで動作する、ゲーム本体から独立したマップ制作ツールです。ビルドや外部ライブラリは不要です。

## 初期版の機能

- 64×64pxグリッドへのタイル配置
- 床／壁・構造／設備／装飾・影の4レイヤー
- レール、食事台、保護室ドア、側壁の0°／45°／90°／135°回転
- 狭い通路用の回転可能な1マスドア
- 高さ1×幅9の横長の窓・カーテン
- 4方向へ回転できる保護室向けの金属製トイレ・金属製手洗い場
- 消しゴム、スポイト、複数選択、まとめて移動・削除、Undo、Redo
- JSON保存・読込・直接編集
- グリッドを除いたPNG書出し
- PCのマウス操作とスマートフォンのタッチ操作
- 添付の部屋見本を基にした「ハード監禁室ver1.0」

## 使い方

1. `index.html`を開きます。
2. パレットからパーツを選びます。
3. 編集領域をクリックまたはタップして配置します。
4. `JSON保存`で再編集用データ、`PNG書出し`で完成画像を保存します。

複数の配置物を動かす場合は、`複数選択`を押して対象を順にタップします。矢印で選択物をまとめて1マス移動できます。`編集中を全選択`では、現在のレイヤーにある配置物を一括選択できます。黄色い選択枠だけを消す場合は`選択枠を解除`、配置物そのものを消す場合は`配置物を削除`を使います。複数配置物の削除時は確認が表示されます。PCでは方向キーによる移動、Deleteキーによる削除にも対応します。

PNG素材へ移行する場合も、タイルIDとJSON形式は維持できます。`editor.js`内の各`draw...`関数を画像描画へ差し替えるだけで、既存マップを作り直す必要はありません。

## JSON形式

```json
{
  "version": 1,
  "name": "部屋名",
  "cellSize": 64,
  "width": 12,
  "height": 11,
  "placements": [
    { "tile": "futon", "x": 4, "y": 3, "layer": "fixture" }
  ]
}
```

座標原点は左上です。`x`は右方向、`y`は下方向へ増えます。
